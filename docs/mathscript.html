MathScript

Mathscript es un superset de javascript echo para poder hacer calculos matematicos tal como mjl pero para hacer paginas web con calculos matematicós mas facil

instalación

~ $ npm install @iancarlos/mathscript #local
~ $ npm install -g @iancarlos/mathscript #global

uso

~ $ msc --init
¿Querés que sea estricto? (sí/no): sí
Ingresá los archivos .ms separados por espacios: src/main.ms
Archivo msconfig.json creado!
~ $ msc 
~ $ node src/main.js
Hola desde MathScript

tambien se pueden usar otras configs ej:

~ $ msc config1.json
~ $ mcs config2.json

aunque de por defecto siempre se usa msconfig.json
